Stage 4

Front-end

take designs from designers and use HTML, CSS, and JavaScript to generate websites

Also: optimize loading speeds, responsive design (how websites look on different screen sizes)

Projects: jQuery library for interactivity and animation, Video games, requests to APIs (application program interfaces)

Back-end

interactions between clients (your device) and servers (computer that processes data), get vs post requests

Mobile programming

iOS and Android, work closely with mobile hardward (e.x. phone's camera)

Projects: responsive design, clean user interface, sync to server, production ready, high performance (using mobile device hardware)

Data Analysis

identify patterns or trends from datasets, use applied statistics and machine learning algorithms, data visualization

Know SQL ('sequel') and R, Python or some other statistical package data modeling involves formally describing the patterns and trends in data to explain the observations or to build functions for predicting future outcomes
