Google Style Guide for Python is full of useful information!!

Put class defintions in one file and then call the class in another file

New vocab words:

  • Class

    Can have both data and methods

  • Instance

    Versions of a class

  • Constructor

    When we create an instance of a class the class' constructor gets called which is essentially the __init__ method inside the class.

  • Self

    • The constructor uses the keyword self which refers to the instance in question

  • Instance Variables

    • Variables associated with a specific instance as defined within __init__. They are unique to an object and can be accessed using the self keyword inside the class or the instance name outside of the class

  • Instance methods

    All of the functions inside the class that are associated with an instance and have the first argument as self.

This information is nicely summarized here:

When creating a new class we define an __init__ function which tells Python to create space in memory to remember details for our new class
It is a good idea to include self as a parameter in your __init__ function! It is kind of like a generic placeholder name for the new instance you create and isn't really counted as a parameter. It is used to define instance variables inside the __init__ function of your new class

Advanced ideas in Object Oriented Programming

Class variables are associated with all instances of a class and are not specific to just one instance they can be accessed like this:

python_filename.Class_name.variable_name

media.Movie.valid_ratings

When you define a variable that is essentially a constant, Google Style Guide suggests you use all caps, e.x.: VALID_RATINGS = ["G", "PG", "PG-13", "R"]

All classes come with preexisting class variables, e.x. __doc__ which holds some documentation. If you define your own class this refers to the """ comments at the start of your class

class Movie(): """ The information you type here = __doc__ """

Can be accessed the same way as class variables above

Other predefined variables include __name__ which refers to the name of the class (e.x. Movie) and __module__ which refers to the name of the file in which the class was defined (e.x. media)
