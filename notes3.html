Take a Break (relevant .py)
Write Python script that opens a browser and plays your favorite song/video every 2 hours, prompting you to take a break

We want our program to:

  1. Wait 2 hours

    Google how to do this! Found on Stackoverflow.com:

    import time

    time.sleep(10) """ 10 = 10 seconds """

  2. Open browser

    Google how to do this! Found:

    import webbrowser

    webbrowser.open("http://whatever.com")

  3. Repeat these steps 3x in a loop
    See .py for how I solved this

Where Does Webbrowser Come From?
Inside Python (that we downloaded) is a file or module called webbroswer

Inside that is a function called open

Same for file called time that contains a variety of useful functions
including ctime which tells us the current time, and sleep which suspends program for a little bit

How do these work? The details are hidden from us, i.e. abstraction. Allows us to focus on the program we want to write and use more complex functions simply by reading their documentation!

Secret Message program (relevant .py)
Renames files!

We want our program to:

  1. Get all the file names

    Google "find filenames in a folder in Python"

    os.listdir() will get you everything that's in a directory - files and directories
    module in Python called os (for operating system) that contains a function listdir which lists everything in a given directory

    takes in a file path and returns list of strings of the filenames:

    import os

    os.listdir(r"C:\Users\bjerickson\Desktop\prank")

  2. For each file, rename it!

    Look through documentation for os module, which function can we use to rename files? rename(), it takes in 'source' or current name of the file and changes it to 'destination' or new name

    use translate function that operates on strings and takes up to 2 arguments: a table which translates one set of characters to another, since we don't have that, we simply input 'None'. The other parameter is a list of characters to remove.

    os.getcwd() retrieves current working directory (cwd)

    os.chdir() changes working directory

What would happen if:
We try to rename a file that does not exist?
We try to rename a file to a name that already exists in the folder?

In both cases we will get an error called an Exception (we will revisit this idea later!)
